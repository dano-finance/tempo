# Tempo – Multi Party Alliance

Tempo is a governance platform built on Cardano, designed to support **Delegated Representatives (DReps)** and ADA holders in forming alliances, coordinating decisions, and ensuring that collective mandates are executed transparently on-chain.  

This repository contains the work for the **Multi Party Alliance** module, developed under a **Project Catalyst Funded Proposal**.

---

## 🎯 Project Goal
- Allow ADA holders and DReps to **create alliances** with defined rules and membership criteria.  
- Provide alliance members with tools to **discuss, propose, and vote** on governance decisions.  
- Ensure that internal alliance mandates are **executed on-chain by DReps**.  
- Deliver the project according to milestones defined in the Catalyst proposal.
