## M3: Main Features Development  
**Delivery:** 13 - Feb 2026  

**Outputs:**  
- Release main features on Testnet including:  
  - ADA holders/DReps can create alliances with defined rules.  
  - ADA holders who meet the criteria can join an alliance.  
  - Alliance members can discuss by creating polls, voting, and commenting.  
  - Voting power of each ADA holder is recorded when they cast their votes.  
- The features have been fully tested by successfully executing the defined test cases.  

**Acceptance criteria:**  
- Testnet deployment accessible for community testing.  
- Features (create alliance, join alliance, polls, voting, commenting, vote recording) function as specified.  
- Test cases executed and passed, with results documented in `/docs/test-report-m3.md`.  

**Evidence:**  
- GitHub repo commits showing implementation in `/src/`.  
- Test report `/docs/test-report-m3.md`.  
- Screenshots or demo video showing features running on Testnet.
